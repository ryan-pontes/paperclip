# v2026.621.0

> Released: 2026-06-21

## Highlights

- **Weekly worktree cleanup script** - Added `scripts/paperclip-cleanup-worktrees.sh` to prune stale agent worktrees, keeping the working tree lean for long-running instances. (#37, NODE-271)

## Fixes

- **Register `claude-accounts` routes in OpenAPI spec** - `GET /api/instance/claude-accounts` and `POST /api/instance/claude-accounts` (added in the multi-account PR) were absent from the spec, causing the `openapi-routes` coverage test to fail deterministically. Both routes and the `apiPrefixes` test map are now updated. (#40)
- **Fix board-chat test mock for `boardAuthService` + `req.actor`** - The board-chat route test lacked a mock for `boardAuthService` (added in #33) and did not set `req.actor` via middleware, causing the feature-flag guard tests to crash with 500 instead of 403. Also updated the deployment-mode guard test to use an explicitly unknown mode now that `authenticated` is a valid mode. (#39, NODE-143)
- **Correct import path in `multi-account.test.ts`** - PR #31 introduced the test importing from `../services/claude-multi-account.js` which does not exist; the module is co-located as `./multi-account.js`. Fixed TS2307 and TS7006. (#38)
